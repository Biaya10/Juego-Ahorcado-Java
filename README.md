# Juego del Ahorcado - Java Swing

Aplicación gráfica desarrollada en Java Swing aplicando los principios de Programación Orientada a Objetos (POO). El proyecto implementa el clásico juego del ahorcado mediante una interfaz gráfica interactiva, permitiendo practicar conceptos de diseño de software y desarrollo de aplicaciones de escritorio.

## Tecnologías Utilizadas

- Java
- Java Swing
- Programación Orientada a Objetos
- NetBeans

## Conceptos Aplicados

- Encapsulación
- Herencia
- Polimorfismo
- Abstracción
- Manejo de eventos
- Interfaces gráficas

## Funcionalidades

- Selección aleatoria de palabras.
- Sistema de categorías.
- Registro de letras correctas e incorrectas.
- Sistema de pistas.
- Representación gráfica del ahorcado.
- Gestión de jugadores.
- Reinicio de partida.

## Estructura del Proyecto

- `Ahorcado.java`: lógica principal del juego.
- `BancoPalabras.java`: administración de palabras y categorías.
- `Jugador.java`: gestión de información del jugador.
- `Palabra.java`: manejo de palabras y validaciones.
- `Principal.java`: punto de inicio de la aplicación.
- `vista/`: componentes gráficos desarrollados con Java Swing.
- `pistas/`: funcionalidades relacionadas con ayudas y pistas.
